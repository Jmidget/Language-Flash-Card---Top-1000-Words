# Language-Flash-Card---Top-1000-Words
Python code for memorizing the top 1000 most commonly used words in a language using flash cards

This code takes the top 1000 most commonly used words in a language (Spanish) and displays it on a 'flash card'. 

If the user can accurately guess the english tranlation of the word before the card 'flips' to show the correct translation, the user clicks a button to remove that term from the list. 

When the application is closed, a list of terms yet to be learned will be saved and used as the word pool the next time the program is run. 

This will continue to happen until the user can correctly match all remaining terms to the corresponding english translation.
